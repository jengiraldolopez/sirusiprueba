Crédito a los fabricantes de recursos de software utilizados por Gea
BlockUI: El plugin de jQuery BlockUI le permite simular el comportamiento síncrono utilizando AJAX, sin bloquear el navegador. Cuando se activa, se evitará que la actividad del usuario en la página (o parte de la página) hasta que se desactive. BlockUI agrega elementos al DOM para darle la apariencia y comportamiento de bloqueo en la interacción con el usuario.
CKEditor: Es un plugin Javascript del tipo WYSIWYG editor HTML, con todas las funciones que necesito. Pero no puede descartarse a CLEditor.
FancyTree: es un plugin JQuery para crear árboles dinámicos con soporte para persistencia, teclado, casillas de verificación, arrastrar y soltar y carga diferida. Cuenta con una página de ejemplos comprensibles y con código fuente sencillo. La documentación es completa y bien estructurada, igual que el tutorial incluido en la wiki del producto. Por último, las versiones del producto se están actualizando constantemente.
Form validation engine: Es un motor de validación para el framework de Javascript jQuery que soporta varios idiomas, reglas de validación predefinidas y definidas por el desarrollador. Las versiones más recientes pueden ser descargadas desde aquí.
FullCalendar: es un plugin de jQuery que ofrece un completo calendario con posibilidades de arrastrar. Utiliza AJAX para gestionar los eventos, se configura fácilmente para utilizar su propio formato de alimentación. Consulte aquí si requiere mostrar información adicional o también dé un vistazo al desarrollo del proyecto.
JQGrid: Este grid es un control javascript basado en Ajax que permite representar datos en la web. Algunas características de JQGrid son: carga de datos XML o formato JSON, configuracion simple, paginación, columnas redimensionables, ordenación de columnas, soporte para imágenes, links o checkboxes, vistas maestro-detalle, subgrids, y edición de celdas.
JQuery: jQuery es un una librería o framework de Javascript que simplifica la manera de construir interfaces de documentos HTML, permitiendo manejar eventos, desarrollar animaciones y agregar interacción con la tecnología AJAX en páginas web. Pulse clic aquí si desea saber más sobre JQuery.
jQuery File Download: es un plugin jQuery multiplataforma que permite descargar archivos mediante Ajax, cosa que normalmente no es posible.
jQuery-MD5: MD5 es un algoritmo de encriptación de datos. Se toma una cadena como entrada y produce un número de n-bits llamado hash. La misma cadena siempre produce el mismo hash, pero si se tiene un hash, no es posible determinar la cadena original.
JQuery UI: Es una librería de componentes para el framework jQuery que provee un conjunto de plugins (drag & drop, sorting, resizing, etc.), efectos visuales (show, hide, toggle, color animation, class manipulation, etc.) y widgets (accordion, date picker, dialog, slider, tabs, etc.).
jQuery UI MultiSelect Widget: Tiene un conjunto de características muy robustas, es rápido, y es mucho más flexible que un select convencional. jQuery UI MultiSelect Widget convierte un simple Select HTML en una lista desplegable de casillas de verificación.
Libchart: Es una libreria PHP que permite generar gráficos de barras o de pastel de forma sencilla. Permite generar gráficos con datos prefijados, así como con datos almacenados en una base de datos.
ODTPhp: Permite generar automáticamente documentos de texto OpenOffice a partir de plantillas. Se puede utilizar directamente en PHP sin necesidad de tener instalado OpenOffice. En el sitio de descarga se puede encontrar una breve información sobre el producto. No obstante OpenTBS y TinyButStrong pueden ser mejores opciones.
PHPExcel: Proporciona un conjunto de clases para el lenguaje de programación PHP, que permite escribir y leer diferentes formatos de archivo, como Excel 2007, PDF, HTML. PHPExcel se basa en el estándar de Microsoft OpenXML y PHP.
PHPMailer: PHPMailer es una clase escrita en PHP que facilita el envío de correo, añadiendo facilidad en el envío de correos con adjuntos, en formato HTML y con diferentes codificaciones, soporte para imagenes embebidas, headers personalizados y además funciona con múltiples servidores de correo.
Plupload: Plupload es una librería JavaScript que brinda la funcionalidad de subir archivos múltiples al servidor, esta librería no solo incorpora Flash para subir archivos, también se pueden HMTL5, Silverlight y BrowserPlus.
Postgres: PostgreSQL es un sistema de gestión de bases de datos objeto-relacional, distribuido bajo licencia BSD y con su código fuente disponible libremente. La estructuras de las bases de datos se puede modelar visualmente mediante PostgreSQL Database Modeler o simplemente pgModeler. Incluso se puede utilizar Libreoffice (dbname=nombredb hostaddr=###.###.###.### port=543# user=usuario password=clave)
Wamp Server: Antes conocido como WAMP, es un completo paquete que permite instalar y configurar fácilmente el servidor Web Apache, el lenguaje de programación PHP y el servidor de base de datos MySQL.
